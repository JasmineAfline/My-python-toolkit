# Python Beginner Toolkit

##  Overview
This project is a beginner-friendly Python toolkit created as part of the Moringa AI Capstone Project. It demonstrates basic programming concepts using simple and practical examples.

##  Objective
The goal of this project is to:
- Learn Python using AI tools
- Build a simple working project
- Document the learning process for beginners

##  Technologies Used
- Python 3.13.12
- VS Code

##  Project Structure
- hello.py → Simple "Hello World" program
- calculator.py → Basic calculator using user input and logic
- Python_Toolkit.md → Full documentation of the learning process

## ▶ How to Run the Project

1. Install Python (version 3.x)
2. Open the project folder in VS Code or terminal
3. Run the files:

### Run Hello World

##  Features
- User input handling
- Basic arithmetic operations
- Conditional logic (if/else)
- Error handling (e.g., division by zero)

##  Learning Highlights
- Understanding Python basics (variables, input/output)
- Building a simple working application
- Using AI to assist learning and debugging
